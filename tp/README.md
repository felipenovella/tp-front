# Temática
Catálogo interactivo de aviones comerciales Boeing desde el 707 (1958) hasta el 787 (2011). Permite explorar los distintos modelos con sus especificaciones técnicas, filtrarlos por tipo de avion, buscarlos en tiempo real y ver detalles de cada uno.

# Items de la consigna resueltos
Requisitos mínimos

✅ 1 página principal — src/pages/index.astro
✅ Al menos 3 secciones — catalogo, estadisticas y sobre el proyecto
✅ Responsive básico — CSS Grid con auto-fill, media queries para mobile (480px) y tablet (768px)
✅ Comportamiento con JavaScript — filtros por tipo, buscador, modal de detalle, dark/light mode

Opcionales

✅ Componentes reutilizables — NavBar.astro, AircraftCard.astro, Modal.astro
✅ Animaciones — fade-in en cards, hover con barra lateral animada
✅ Dark mode
